# BKey UI Kit

`bkey_uikit` is a Flutter package that packages the **BMoni design system** into a reusable library — colours, typography, theme, and ~30 production-ready widgets. Any Bkey Flutter app can depend on it and get a consistent look and feel without copying code.

## Design principles

- **No app coupling.** Components have zero dependencies on app-level state management, routing, localisation, or services.
- **Token-driven.** All colours and text styles come from `BMoniColors` and `BMoniTextStyles` — override at the theme level, not in widget code.
- **Const-first.** Every widget that can use a `const` constructor does so, keeping rebuild costs minimal.
- **Single import.** Everything is exported from `package:bkey_uikit/bkey_uikit.dart`.
