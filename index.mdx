# BKey UI Kit

`bkey_uikit` is a Flutter package that packages the **BMoni design system** into a reusable library — colours, typography, theme, and reusable UI primitives. Any Bkey Flutter app can depend on it and get a consistent look and feel without copying code.

## What's included

| Layer | Contents |
|---|---|
| **Design Tokens** | `BMoniColors` — full colour palette + semantic light/dark tokens |
| | `BMoniTextStyles` — complete type scale (display, heading, body, label) |
| | `BMoniTheme` — Flutter `ThemeData` factory + `TextTheme` wiring |
| **Enums** | Button, text-field, display, and status enums shared by all components |
| **Components** | Buttons, text fields, avatars, toasts, bottom sheets, empty states, and layout primitives such as `SectionHeader`, `ActivitySectionCard`, `PageDotsIndicator`, and `BouncingPageViewPhysics` |
| **Fonts** | _Rethink Sans_ (400–800, regular + italic) bundled in `assets/fonts/` |

## Latest updates

- Added `SectionHeader` for reusable section titles with optional trailing actions
- Added `ActivitySectionCard` for bordered activity and dashboard sections
- Added `PageDotsIndicator` for paged UI such as wallet or card carousels
- Added `BouncingPageViewPhysics` for package-safe page-view bounce behavior
- Confirmed package boundaries: `bkey_uikit` owns generic UI only; feature-aware composition now lives in packages such as `bmoni_embedded_wallets_cards`

## Looking for wallet cards?

If you are trying to find the wallet/card UI layer:

- generic wallet/card layout primitives are documented under [Wallet & card layouts](/components/wallet-card-layouts)
- full wallet/card-aware composition lives in `bmoni_embedded_wallets_cards`
- backend HTTP endpoints are documented separately under [REST API: Wallets](/api/wallets) and [REST API: Cards](/api/cards)

## Design principles

- **No app coupling.** Components have zero dependencies on app-level state management, routing, localisation, or services.
- **Token-driven.** All colours and text styles come from `BMoniColors` and `BMoniTextStyles`.
- **Const-first.** Every widget that can use a `const` constructor does so.
- **Single import.** Everything is exported from `package:bkey_uikit/bkey_uikit.dart`.
