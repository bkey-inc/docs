## BMoniButton

The main button component. Four variants, five sizes, icon support, loading state.

```dart
BMoniButton(
  onPressed: () {},
  text: 'Confirm',
)
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `onPressed` | `VoidCallback?` | — | Tap callback. `null` renders disabled. |
| `text` | `String` | — | Button label. |
| `variant` | `BMoniButtonVariant` | `primary` | Visual style. |
| `size` | `BMoniButtonSize` | `medium` | Height/padding tier. |
| `isLoading` | `bool` | `false` | Shows spinner, blocks taps. |
| `isDisabled` | `bool` | `false` | Disabled visual state. |
| `icon` | `IconData?` | — | Optional icon. |
| `iconPosition` | `BMoniButtonIconPosition` | `leading` | Before or after label. |
| `width` | `double?` | — | Fixed width. |
| `backgroundColor` | `Color?` | — | Override background. |
| `foregroundColor` | `Color?` | — | Override text + icon colour. |
| `outlineColor` | `Color?` | — | Override border colour. |
| `textColor` | `Color?` | — | Override text colour only. |

### Variants

**Primary** — filled with `BMoniColors.brand600`. Main call-to-action.
```dart
BMoniButton(onPressed: () {}, text: 'Save')
BMoniButton.primary(onPressed: () {}, text: 'Save')
```

**Secondary** — transparent background, brand-coloured text.
```dart
BMoniButton.secondary(onPressed: () {}, text: 'Learn more')
```

**Outline** — transparent with border. Low-priority actions.
```dart
BMoniButton.outline(onPressed: () {}, text: 'Cancel')
```

**Ghost** — no background or border. Dismissal/skip.
```dart
BMoniButton.ghost(onPressed: () {}, text: 'Skip')
```

### Sizes

| Size | Height |
|---|---|
| `extraSmall` | 32 px |
| `small` | 40 px |
| `medium` | 48 px (default) |
| `large` | 56 px |
| `custom` | 56 px + extra padding |

### Loading state

```dart
BMoniButton(
  onPressed: _submit,
  text: 'Submitting',
  isLoading: _isLoading,
)
```

### Full-width

```dart
BMoniButton(
  onPressed: () {},
  text: 'Continue',
  width: double.infinity,
)
```

## UtilityButton

A compact square icon button.

```dart
UtilityButton(
  onTap: () {},
  icon: const Icon(Icons.add, color: Colors.white),
)
```

| Prop | Type | Default |
|---|---|---|
| `onTap` | `VoidCallback` | — |
| `icon` | `Widget` | `Icon(Icons.add)` |
| `size` | `double` | `36.0` |
| `backgroundColor` | `Color` | `BMoniColors.brand600` |
| `outlineColor` | `Color` | `Colors.transparent` |
| `borderRadius` | `double` | `8.0` |
