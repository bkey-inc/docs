## CustomAppBar

A `PreferredSizeWidget` that wraps Flutter's `AppBar` with BMoni styling — no elevation, transparent background, and support for a back button or close button.

```dart
CustomAppBar(title: 'Settings')

// With a custom leading action
CustomAppBar(
  title: 'Payment',
  showCloseButton: true,
  backPressed: () => Navigator.pop(context),
)

// With a custom title widget
CustomAppBar(
  titleWidget: Row(children: [avatar, const SizedBox(width: 8), nameText]),
  actions: [IconButton(icon: const Icon(Icons.more_vert), onPressed: _menu)],
)
```

<ParamField path="title" type="String?">Plain string title. Either `title` or `titleWidget` is required.</ParamField>
<ParamField path="titleWidget" type="Widget?">Custom title widget.</ParamField>
<ParamField path="showBackButton" type="bool" default="true" />
<ParamField path="showCloseButton" type="bool" default="false">Shows an ✕ instead of the back arrow.</ParamField>
<ParamField path="backPressed" type="VoidCallback?">Custom back/close handler. Defaults to `Navigator.maybePop`.</ParamField>
<ParamField path="actions" type="List\<Widget\>" default="[]" />
<ParamField path="centerTitle" type="bool" default="true" />
<ParamField path="backgroundColor" type="Color?" />

## SwipeableActionRow

A list item that reveals an action button when swiped left. Multiple items sharing the same `openItemId` notifier auto-close when another is opened.

```dart
final _openItemId = ValueNotifier<String?>(null);

SwipeableActionRow(
  itemId: transaction.id,
  openItemId: _openItemId,
  actionColor: BMoniColors.error600,
  actionIcon: const Icon(Icons.delete, color: Colors.white),
  onActionPressed: () => _delete(transaction),
  onTap: () => _openDetails(transaction),
  child: TransactionRow(transaction: transaction),
)
```

<ParamField path="itemId" type="String" required>Unique identifier for this row.</ParamField>
<ParamField path="openItemId" type="ValueNotifier\<String?\>" required>Shared notifier. Only one row is open at a time.</ParamField>
<ParamField path="child" type="Widget" required>The row content shown when not swiped.</ParamField>
<ParamField path="actionWidth" type="double" default="72.0" />
<ParamField path="actionIcon" type="Widget" default="Icon(Icons.more_horiz)" />
<ParamField path="actionColor" type="Color" default="Colors.purple" />
<ParamField path="onActionPressed" type="VoidCallback?" />
<ParamField path="onTap" type="VoidCallback?" />

## TierBenefitItem

A single benefit line item used in tier/subscription cards. Shows a check icon, description, and an optional extra label.

```dart
TierBenefitItem(
  description: 'Free international transfers',
  isCurrentTier: true,
  isAvailable: true,
)

TierBenefitItem(
  description: 'Priority support',
  isCurrentTier: false,
  isAvailable: false, // renders in grey
)
```

<ParamField path="description" type="String" required />
<ParamField path="isCurrentTier" type="bool" required>White text when `true`; grey text when `false`.</ParamField>
<ParamField path="isAvailable" type="bool" default="true">Dims the item when `false`.</ParamField>
<ParamField path="extraText" type="String?">Additional label shown after the description (e.g. a tier badge name).</ParamField>

## CurrencyOption

A tappable currency row with a flag image, currency code, and a trailing arrow. Used inside currency-selection sheets.

```dart
CurrencyOption(
  currency: {'flag': 'assets/images/flags/ng.png', 'code': 'NGN'},
  isSelected: selectedCurrency == 'NGN',
  onTap: () => setState(() => selectedCurrency = 'NGN'),
)
```

<ParamField path="currency" type="Map\<String, String\>" required>Must contain `'flag'` (asset path) and `'code'` (display string).</ParamField>
<ParamField path="isSelected" type="bool" required />
<ParamField path="onTap" type="VoidCallback" required />
