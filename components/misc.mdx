## CustomAppBar

`PreferredSizeWidget` wrapping Flutter's `AppBar` with BMoni styling.

```dart
CustomAppBar(title: 'Settings')

CustomAppBar(
  title: 'Payment',
  showCloseButton: true,
  backPressed: () => Navigator.pop(context),
  actions: [IconButton(icon: const Icon(Icons.more_vert), onPressed: _menu)],
)
```

| Prop | Type | Default |
|---|---|---|
| `title` | `String?` | — |
| `titleWidget` | `Widget?` | — |
| `showBackButton` | `bool` | `true` |
| `showCloseButton` | `bool` | `false` |
| `backPressed` | `VoidCallback?` | `Navigator.maybePop` |
| `actions` | `List<Widget>` | `[]` |
| `centerTitle` | `bool` | `true` |
| `backgroundColor` | `Color?` | — |

## SwipeableActionRow

List item that reveals an action button on left swipe. Multiple items share an `openItemId` notifier — only one is open at a time.

```dart
final _openItemId = ValueNotifier<String?>(null);

SwipeableActionRow(
  itemId: transaction.id,
  openItemId: _openItemId,
  actionColor: BMoniColors.error600,
  actionIcon: const Icon(Icons.delete, color: Colors.white),
  onActionPressed: () => _delete(transaction),
  onTap: () => _openDetails(transaction),
  child: TransactionRow(transaction: transaction),
)
```

| Prop | Type | Default |
|---|---|---|
| `itemId` | `String` | — |
| `openItemId` | `ValueNotifier<String?>` | — |
| `child` | `Widget` | — |
| `actionWidth` | `double` | `72.0` |
| `actionIcon` | `Widget` | `Icon(Icons.more_horiz)` |
| `actionColor` | `Color` | `Colors.purple` |

## BouncingPageViewPhysics

Page-view scroll physics that preserve bounce feedback at the boundaries.

```dart
PageView(
  physics: const BouncingPageViewPhysics(),
  children: const [Placeholder(), Placeholder()],
)
```

No props beyond the inherited `ScrollPhysics.parent`.
