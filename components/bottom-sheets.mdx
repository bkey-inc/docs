## BMoniBottomSheet

Static helper wrapping Flutter's `showModalBottomSheet` with BMoni styling.

```dart
await BMoniBottomSheet.show(
  context: context,
  child: const MySheetContent(),
);
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `context` | `BuildContext` | — | — |
| `child` | `Widget` | — | — |
| `isDismissible` | `bool` | `true` | — |
| `enableDrag` | `bool` | `true` | — |
| `height` | `double?` | — | Fixed height. |
| `isScrollable` | `bool` | `false` | Sets `isScrollControlled: true`. |
| `backgroundColor` | `Color?` | — | — |

## BMoniTitledBottomSheet

Ready-made sheet chrome: drag handle, title, and close button.

```dart
BMoniBottomSheet.show(
  context: context,
  isScrollable: true,
  child: BMoniTitledBottomSheet(
    title: 'Select currency',
    children: [
      // your content
    ],
  ),
);
```

| Prop | Type | Default |
|---|---|---|
| `title` | `String` | — |
| `children` | `List<Widget>` | — |
| `onClose` | `VoidCallback?` | `Navigator.pop` |

## BMoniOptionsBottomSheet

Pre-built options list sheet.

```dart
BMoniOptionsBottomSheet(
  title: 'More options',
  options: [
    BMoniBottomSheetOption(
      icon: 'assets/svgs/edit.svg',
      title: 'Edit',
      onTap: () => Navigator.pop(context, 'edit'),
    ),
  ],
)
```

## SelectorBottomSheet

Scrollable list-picker. Generic over any item type.

```dart
SelectorBottomSheet<Country>(
  items: availableCountries,
  label: (c) => c.name,
  value: (c) => c.id,
  icon: (c) => c.flagAsset,
  showIcon: true,
  title: 'Select country',
)
```

| Prop | Type | Default |
|---|---|---|
| `items` | `List<T>` | — |
| `label` | `String Function(T)` | — |
| `value` | `Object? Function(T)` | — |
| `icon` | `String? Function(T)?` | — |
| `showIcon` | `bool` | `false` |
| `title` | `String?` | — |
| `selectOptionLabel` | `String` | `'Select Option'` |

## SearchableSelectorBottomSheet

Same as `SelectorBottomSheet` plus a search field.

```dart
SearchableSelectorBottomSheet<Bank>(
  items: banks,
  label: (b) => b.name,
  value: (b) => b.isSelected,
  searchHint: 'Search banks...',
  filterFunction: (query, bank) =>
      bank.name.toLowerCase().contains(query.toLowerCase()),
)
```

Additional props:

| Prop | Type | Default |
|---|---|---|
| `searchHint` | `String` | — |
| `filterFunction` | `bool Function(String, T)` | — |
| `noResultsLabel` | `String` | `'No results found'` |
