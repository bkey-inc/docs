## BMoniToast

A notification banner with four variants: default, warning, success, error.

```dart
BMoniToast(
  message: 'Payment of $50 sent.',
  variant: BMoniToastVariant.success,
  title: 'Sent!',
)
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `message` | `String` | — | Main text. |
| `variant` | `BMoniToastVariant` | `default_` | Colour/icon style. |
| `title` | `String?` | — | Bold heading. |
| `trailing` | `Widget?` | — | Custom trailing widget. |
| `toastType` | `BMoniToastType` | `actionable` | `actionable` = chevron, `closeable` = X. |
| `showTrailing` | `bool` | `true` | — |
| `onTap` | `VoidCallback?` | — | — |

### Convenience constructors

```dart
BMoniToast.success(message: 'Done!')
BMoniToast.error(message: 'Something went wrong')
BMoniToast.warning(message: 'Low balance')
BMoniToast.info(message: 'New feature available')
```

## BMoniToastOverlay

Floating toast displayed above the current page. Swipe-to-dismiss, auto-hides.

```dart
BMoniToastOverlay.showSuccess(
  context: context,
  message: 'Transfer complete',
  title: 'Success',
  duration: const Duration(seconds: 4),
);
```

| Method | Description |
|---|---|
| `showDefault(...)` | Info-style toast |
| `showWarning(...)` | Warning toast |
| `showSuccess(...)` | Success toast |
| `showError(...)` | Error toast |
| `hide()` | Dismiss programmatically |

All methods accept: `context`, `message`, `title?`, `duration`, `isBottom`, `trailing?`, `onTap?`.

Only one toast is shown at a time — showing a new one dismisses the existing one.
