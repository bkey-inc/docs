## EmptyState

Centred illustration + message with an optional action button.

```dart
EmptyState(
  svgAsset: 'assets/svgs/empty_box.svg',
  message: 'No transactions yet',
  subtitle: 'Your history will appear here.',
)
```

```dart
// With action
EmptyState(
  svgAsset: 'assets/svgs/empty_friends.svg',
  message: 'No contacts yet',
  buttonText: 'Add a contact',
  onButtonPressed: _openAddContact,
)
```

| Prop | Type | Default |
|---|---|---|
| `svgAsset` | `String` | — |
| `message` | `String` | — |
| `subtitle` | `String?` | — |
| `buttonText` | `String?` | — |
| `onButtonPressed` | `VoidCallback?` | — |
| `imageWidth` | `double?` | `64` |
| `imageHeight` | `double?` | `64` |
| `titleSize` | `BodySize?` | `BodySize.small` |
| `textColor` | `Color?` | — |
| `backgroundColor` | `Color?` | — |
| `height` | `double?` | — |

## FailureWidget

Error card with icon, title, body, optional rejection list, and retry button.

```dart
FailureWidget(
  title: 'Verification failed',
  body: 'We could not verify your identity.',
  rejectionReasons: ['Photo was blurry'],
  onRetryPressed: _retry,
)
```

| Prop | Type | Default |
|---|---|---|
| `title` | `String` | — |
| `body` | `String` | — |
| `rejectionReasons` | `List<String>` | `[]` |
| `onRetryPressed` | `VoidCallback?` | — |
| `buttonText` | `String` | `'Try again'` |
| `issuesFoundLabel` | `String` | `'Issues found:'` |

## InProgressWidget

Pending state card with hourglass icon.

```dart
InProgressWidget(
  title: 'Verification in progress',
  body: 'This usually takes 1–2 minutes.',
)
```

| Prop | Type |
|---|---|
| `title` | `String` |
| `body` | `String` |
| `backgroundColor` | `Color?` |

## ProgressLoaderWidget

Compact circular loading indicator in a branded container.

```dart
const ProgressLoaderWidget()
```

No props. Fixed 80×80 px container with `CupertinoActivityIndicator`.

## InfoCard

Horizontal card with icon and message for tips/warnings.

```dart
InfoCard(
  message: 'Transfers can take up to 3 business days.',
)

InfoCard(
  title: 'Daily limit reached',
  message: 'You have reached your daily limit of $500.',
  icon: Icons.warning_amber_rounded,
)
```

| Prop | Type | Default |
|---|---|---|
| `message` | `String` | — |
| `title` | `String?` | — |
| `icon` | `IconData` | `Icons.info` |
| `iconColor` | `Color?` | `bg.warningSolid` |
| `backgroundColor` | `Color?` | — |
| `child` | `Widget?` | — |
