## ProfileAvatar

Circular avatar. Shows a cached network image or falls back to initials.

```dart
ProfileAvatar(
  imageUrl: 'https://cdn.example.com/avatar.jpg',
  name: 'John Doe',
  size: 48,
)
```

| Prop | Type | Description |
|---|---|---|
| `imageUrl` | `String?` | Image URL. `null` always shows initials. |
| `name` | `String` | Used to derive initials. |
| `size` | `double` | Diameter in logical pixels. |

## ChatAvatar

Avatar for chat threads. Uses a deterministic background colour per user.

```dart
ChatAvatar(
  imageUrl: member.avatarUrl,
  name: member.displayName,
  userId: member.id,
  size: 40,
)
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `imageUrl` | `String?` | — | — |
| `name` | `String` | — | Initial fallback. |
| `userId` | `String?` | — | Drives background colour. |
| `size` | `double` | `48` | — |
| `radius` | `double?` | — | Clips to rounded rect when set. |

## ProfileImageWidget

Flexible image widget — supports network URLs, asset paths, and file paths.

```dart
ProfileImageWidget(
  imageUrl: 'https://cdn.example.com/photo.jpg',
  radius: 32,
)
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `imageUrl` | `String` | — | Network, asset, or file path. |
| `radius` | `double?` | — | Half the diameter. |
| `width` / `height` | `double?` | — | Explicit size. |
| `placeholder` | `Widget?` | — | Loading widget. |
| `errorWidget` | `Widget?` | — | Error widget. |
| `onTap` | `VoidCallback?` | — | — |
| `fit` | `BoxFit?` | `BoxFit.cover` | — |

## FileUploadWidget

Dotted-border upload area. Handles empty, uploading, and uploaded states.

```dart
FileUploadWidget(
  onTap: _pickFile,
  onRemove: _removeFile,
  fileBytes: _selectedFileBytes,
  fileName: _selectedFileName,
  placeholderText: 'Tap to upload your ID',
  errorMessage: _error,
)
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `onTap` | `VoidCallback` | — | Opens file picker. |
| `fileBytes` | `Uint8List?` | — | Set to show uploaded state. |
| `fileName` | `String?` | — | Display name of uploaded file. |
| `onRemove` | `VoidCallback?` | — | Shows remove button. |
| `errorMessage` | `String?` | — | Shown below in red. |
| `placeholderText` | `String` | `'Tap to upload'` | — |
| `uploadedText` | `String` | `'Uploaded!'` | — |
| `isLoading` | `bool` | `false` | — |
