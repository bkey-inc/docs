## Overview

Instead of manually applying `BMoniTextStyles` everywhere, use the semantic text widgets. They accept human-readable `size` and `weight` enums and apply the correct `TextStyle` automatically.

```dart
HeadingText('Section title', level: 3)
BodyText('Read more about this topic', size: BodySize.small)
LabelText('Form label', size: LabelSize.large, weight: LabelWeight.medium)
```

---

## AmountText

Displays a financial amount with an optional currency symbol and configurable decimal places.

## ReadMoreText

A "read more / show less" toggle for long text passages.

```dart
ReadMoreText(
  'This is a very long description that will be truncated…',
  trimLines: 3,
  trimMode: TrimMode.Line,
  colorClickableText: BMoniColors.brand400,
)
```

<ParamField path="data" type="String" required />
<ParamField path="trimMode" type="TrimMode" default="Length">`Length` or `Line`</ParamField>
<ParamField path="trimLength" type="int" default="240">Characters shown before truncation (TrimMode.Length).</ParamField>
<ParamField path="trimLines" type="int" default="2">Lines shown before truncation (TrimMode.Line).</ParamField>
<ParamField path="trimCollapsedText" type="String" default="'read more'">Link text when collapsed.</ParamField>
<ParamField path="trimExpandedText" type="String" default="'show less'">Link text when expanded.</ParamField>
<ParamField path="colorClickableText" type="Color?" />
<ParamField path="onExpandChanged" type="void Function(bool)?" />
<ParamField path="isExpandable" type="bool" default="true">Set to `false` to use `onLinkTap` instead of toggling.</ParamField>
