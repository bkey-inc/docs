---
title: Text widgets
description: Typed text widgets that automatically apply the correct BMoni style.
---

Use these instead of applying `BMoniTextStyles` manually.

```dart
HeadingText('Section title', level: 3)
BodyText('Body copy', size: BodySize.medium)
LabelText('Label', size: LabelSize.large, weight: LabelWeight.medium)
```

## DisplayText

Levels 1–6 map to D1–D6.

```dart
DisplayText('Hero', level: 5, weight: DisplayWeight.bold)
```

| Prop | Type | Default |
|---|---|---|
| `text` | `String` | — |
| `level` | `int` | `1` (1–6) |
| `weight` | `DisplayWeight` | `bold` |
| `color` | `Color?` | — |
| `textAlign` | `TextAlign?` | — |
| `maxLines` | `int?` | — |
| `overflow` | `TextOverflow?` | — |

## AmountText

```dart
AmountText(1234.50, currency: '\$', decimalPlaces: 2)
// Renders: $1234.50
```

| Prop | Type | Default |
|---|---|---|
| `amount` | `double` | — |
| `currency` | `String` | `'$'` |
| `showCurrency` | `bool` | `true` |
| `decimalPlaces` | `int` | `2` |
| `style` | `TextStyle?` | — |

## ReadMoreText

"Read more / show less" toggle for long text.

```dart
ReadMoreText(
  'This is a very long description...',
  trimLines: 3,
  trimMode: TrimMode.line,
  colorClickableText: BMoniColors.brand400,
)
```

| Prop | Type | Default |
|---|---|---|
| `data` | `String` | — |
| `trimMode` | `TrimMode` | `length` |
| `trimLength` | `int` | `240` |
| `trimLines` | `int` | `2` |
| `trimCollapsedText` | `String` | `'read more'` |
| `trimExpandedText` | `String` | `'show less'` |
| `colorClickableText` | `Color?` | — |
| `isExpandable` | `bool` | `true` |
