## BMoniTextFormField

A `TextFormField` wrapper with BMoni design system styling.

```dart
BMoniTextFormField(
  label: 'Email address',
  hintText: 'you@example.com',
  keyboardType: TextInputType.emailAddress,
)
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `String?` | — | Label above the field. |
| `hintText` | `String?` | — | Placeholder text. |
| `helperText` | `String?` | — | Helper text below field. |
| `errorText` | `String?` | — | Error message (turns border red). |
| `variant` | `BMoniTextFieldVariant` | `filled` | `filled` or `outlined`. |
| `size` | `BMoniTextFieldSize` | `medium` | `small`, `medium`, or `large`. |
| `prefixIcon` | `Widget?` | — | Widget at input start. |
| `prefixText` | `String?` | — | Static text prefix. |
| `suffixIcon` | `Widget?` | — | Widget at input end. |
| `obscureText` | `bool` | `false` | Hide input for passwords. |
| `controller` | `TextEditingController?` | — | External controller. |
| `validator` | `String? Function(String?)?` | — | FormField validator. |

### Convenience constructors

```dart
// Filled (dark background)
BMoniTextFormField.filled(label: 'Name', hintText: 'John Doe')

// Outlined (transparent with border)
BMoniTextFormField.outlined(label: 'Search', hintText: 'Type to search...')
```

## BMoniTextAreaField

Multi-line input with a live grapheme character counter. Handles emoji correctly.

```dart
BMoniTextAreaField(
  controller: _noteController,
  label: 'Add a note (optional)',
  hintText: 'Add a note here',
  maxLength: 60,
  maxLines: 3,
)
```

| Prop | Type | Default | Description |
|---|---|---|---|
| `controller` | `TextEditingController` | — | Required. Drives the counter. |
| `label` | `String` | `'Add a note (optional)'` | Label above field. |
| `hintText` | `String` | `'Add a note here'` | Placeholder. |
| `maxLength` | `int` | `60` | Max grapheme clusters. |
| `maxLines` | `int` | `3` | Visible rows. |
| `readOnly` | `bool` | `false` | Disable editing. |
| `onChanged` | `ValueChanged<String>?` | — | Called on change. |
