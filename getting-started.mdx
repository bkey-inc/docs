## Installation

### 1. Add the dependency

**Monorepo / local path:**

```yaml
# pubspec.yaml
dependencies:
  bkey_uikit:
    path: ../bkey_uikit
```

**Git (current development branch):**

```yaml
# pubspec.yaml
dependencies:
  bkey_uikit:
    git:
      url: https://github.com/bkey-inc/Bkey-mobile-uikit.git
      ref: extract-components
```

### 2. Run pub get

```bash
flutter pub get
```

### 3. Apply the theme

```dart
import 'package:bkey_uikit/bkey_uikit.dart';
import 'package:flutter/material.dart';

void main() {
  runApp(
    MaterialApp(
      theme: BMoniTheme.darkTheme(),
      home: const MyHomePage(),
    ),
  );
}
```

## Single import

All public API is available from one import:

```dart
import 'package:bkey_uikit/bkey_uikit.dart';
```

## Package structure

```
lib/
  bkey_uikit.dart        ← single barrel export
  src/
    tokens/
      colors.dart        ← BMoniColors
      text_styles.dart   ← BMoniTextStyles
      bmoni_theme.dart   ← BMoniTheme
    enums/               ← shared enumerations
    components/          ← ~30 widget files
assets/
  fonts/                 ← Rethink Sans TTF files
```
