## Overview

`BMoniColors` is a static class that holds every colour token used by the design system. It is organised in two layers:

- **Primitive tokens** — raw hex values (e.g. `brand500`, `grey300`)
- **Semantic tokens** — purpose-scoped groups that automatically follow light or dark mode

## Usage

```dart
import 'package:bkey_uikit/bkey_uikit.dart';

// Primitive
Container(color: BMoniColors.brand500)
Container(color: BMoniColors.error600)

// Semantic — theme-aware via BuildContext extension
Container(color: context.colors.bg.greySubtle)
Text('Label', style: TextStyle(color: context.colors.text.greyDefault))
```

## Semantic tokens

Semantic tokens are returned by the `context.colors` extension and resolve to the correct primitive based on the current theme brightness.

```dart
// Available via BuildContext
context.colors.bg      // BMoniBackgroundColors
context.colors.border  // BMoniBorderColors
context.colors.text    // BMoniTextColors
```
