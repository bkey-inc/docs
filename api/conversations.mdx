## Conversations

### Send image message to a channel

```http
POST /conversations/channels/{channelId}/messages/image
```

Uploads a chat attachment (image, video, or document) to S3 and sends it to the specified Stream channel.


**Path parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `channelId` | `string` | Yes |  |

**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `sent` | `boolean` | Whether the message was sent |
| `channelId` | `string` | Target channel ID |
| `channelType` | `messaging \| team` | Resolved Stream channel type |
| `messageId` | `string` | Created Stream message ID |
| `imageUrl` | `string` | Image URL sent in attachment |


---


### Get paginated image media for a channel

```http
GET /conversations/channels/{channelId}/media
```

Returns image attachments from channel messages. Media is sourced from Stream with cursor pagination.


**Path parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `channelId` | `string` | Yes |  |

**Query parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `limit` | `integer` | No |  |
| `next` | `string` | No |  |

**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `items` | `object[]` | Media items in the channel |
| `next` | `string \| null` | Cursor for the next page, null when no more pages |

### Archive a direct or group chat for current user

```http
POST /conversations/channels/archive
```

Archives channel for current user only. Used by frontend to move chat into Archived tab.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `channelType` | `messaging \| team` | Yes |  |
| `channelId` | `string` | Yes |  |
| `clearHistory` | `boolean` | No |  |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `archived` | `boolean` | Archive status for current user |
| `channelType` | `messaging \| team` | Stream channel type for archived or unarchived channel |
| `channelId` | `string` | Stream channel ID |


---

## Conversations - Chat

### Create or get direct message channel

```http
POST /conversations/chat/channels/direct
```

Creates 1-on-1 chat with another active user (blocked users are not allowed).


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `contactUserId` | `string` | Yes |  |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `channelId` | `string` | Channel ID (e.g., "user1-user2") |
| `channelCid` | `string` | Full channel CID (e.g., "messaging:user1-user2") |

## Conversations - Video

### Create or get a direct 1-on-1 call with a contact

```http
POST /conversations/video/calls/direct
```

Returns deterministic callId/callCid for Stream Video. Only works with mutual contacts.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `contactUserId` | `string` | Yes |  |
| `callType` | `default \| audio \| video` | No |  |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `callId` | `string` | Call ID (deterministic, Stream-safe) |
| `callCid` | `string` | Full call CID (e.g., "default:callId") |
