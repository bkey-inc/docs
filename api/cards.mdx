## Cards

### Create a card (charges creation fee first)

```http
POST /api/cards
```

Initiates card creation by charging the creation fee on-chain. Returns a signature request. After signing, the card is created automatically via the card service.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `cardName` | `string` | Yes |  |
| `cardColor` | `string` | Yes |  |
| `currency` | `NGN \| USD` | Yes | Currency of the card (NGN or USD) |
| `type` | `physical \| virtual` | Yes | Card type (physical or virtual) |
| `walletId` | `string` | Yes |  |
| `nin` | `string` | No |  |
| `ninIssueDate` | `string` | No |  |
| `ninExpiryDate` | `string` | No |  |
| `deliveryAddress` | `string` | No |  |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `feeAmount` | `string` |  |
| `feeCurrency` | `string` |  |
| `workflowId` | `string` |  |

## Card Fund

### Initiate card funding from wallet

```http
POST /card-fund
```

Starts the card funding process. Offramps funds from user wallet to company account, then credits the card via Blusalt. Returns a signature request that must be signed by the user.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `cardId` | `string` | Yes |  |
| `walletId` | `string` | Yes |  |
| `amount` | `number` | Yes |  |
| `currency` | `NGN \| USD` | Yes | Currency of the card (NGN or USD) |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `method` | `evm \| solana` |  |
| `walletIndex` | `number` |  |
| `workflowId` | `string` |  |
| `hashToSign` | `string` |  |
| `payload` | `string` |  |
| `deadline` | `string` |  |

## Card Withdraw

### Initiate card withdrawal to wallet

```http
POST /card-withdraw
```

Starts the card withdrawal process. Debits the card via Blusalt and transfers stablecoins from the buffer pool to the user wallet. No signature required.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `cardId` | `string` | Yes |  |
| `walletId` | `string` | Yes |  |
| `amount` | `number` | Yes |  |
| `currency` | `NGN \| USD` | Yes | Currency of the card (NGN or USD) |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `transactionId` | `string` | Transaction ID for tracking |
| `workflowId` | `string` | Temporal workflow ID |
| `status` | `PENDING \| PROCESSING` | Initial status |
| `message` | `string` | Status message |
