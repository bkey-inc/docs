## Offramp

### Create offramp to Nigerian bank account (deprecated)

```http
POST /offramp/nigeria
```

Creates an offramp transaction to a Nigerian bank account. DEPRECATED: Use POST /v2/offramp/nigeria instead for quote information.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `sourceWalletId` | `string` | Yes | Source wallet ID to offramp from |
| `bankAccountId` | `string` | Yes | Bank account ID to receive funds |
| `fromAmount` | `string` | Yes | Amount of USDB to offramp |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `method` | `evm \| solana` |  |
| `walletIndex` | `number` |  |
| `workflowId` | `string` |  |
| `hashToSign` | `string` |  |
| `payload` | `string` |  |
| `deadline` | `string` |  |

## Offramp - China

### Get China payout options

```http
GET /offramp/china/options
```

Returns available options for bank names, remittance purposes, and destination currencies for China payout


---


### Create and submit China offramp in one request

```http
POST /offramp/china/create-and-submit
```

Creates a China offramp draft, uploads invoices (1-5 required), and submits it for processing in a single atomic operation. If any step fails, all changes are rolled back. Returns signature request and quote.


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` | Quote details |


---


### Create and submit China offramp V2 (uses counterparties)

```http
POST /offramp/china/v2/create-and-submit
```

Creates a China offramp transaction using pre-created counterparties. The receiver counterparty ID references an existing beneficiary with bank details. This reduces redundancy and enables beneficiary reuse.

Prerequisites:
- Sender counterparty: POST /conduit/counterparties/sender
- Receiver counterparty: POST /conduit/counterparties/receiver


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` | Quote details |

## Offramp - China NGN

### Create and submit China NGN/CNY offramp

```http
POST /offramp/china-ngn/create-and-submit
```

Creates a China NGN/CNY offramp using a saved RMB receiver. Pass `receiverId` (format: `rmb_{uuid}`, obtained from `POST /conduit/counterparties/receiver` with `destinationCurrency: RMB`). Beneficiary details are loaded from the receiver; no need to send account name, bank, or phone. Requires 1–5 invoice files (JPEG, PNG, or PDF). Returns a signature request for the on-chain transfer.


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` |  |
