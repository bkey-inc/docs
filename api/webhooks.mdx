## Conduit Webhooks

### Receive Conduit webhook events

```http
POST /conduit/webhook
```

Endpoint for receiving webhook events from Conduit for transaction and counterparty status updates. Requests are authenticated via HMAC SHA-256 signature in the conduit-signature header.


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Indicates successful webhook processing |
