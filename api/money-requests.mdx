## Money Requests

### Create a new money request

```http
POST /user/money-requests
```


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `toWalletId` | `string` | Yes | The wallet ID to request money to |
| `requesteeId` | `string` | Yes | The user ID to request money from |
| `amount` | `number` | Yes | The amount to request |
| `note` | `string` | No | Optional note for the money request |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Operation success status |
| `moneyRequest` | `object` | Created money request details |


---


### Get money requests for the authenticated user

```http

```


**Query parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `status` | `PENDING \| APPROVED \| REJECTED \| CANCELLED \| PARTIALLY_FULFILLED` | No | Filter money requests by status |

**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |

## Money Requests V2

### Create a money request

```http
POST /v2/money-requests
```


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `requesteeId` | `string` | Yes | The user ID to request money from |
| `amount` | `string` | Yes | The amount to request as a decimal string |
| `currency` | `USD \| NGN \| CAD \| EUR \| GBP \| MXN` | Yes | The currency for the money request |
| `note` | `string` | No | Optional note for the money request |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | `string` | Money request unique ID |
| `amount` | `string` | Requested amount as decimal string |
| `currency` | `USD \| NGN \| CAD \| EUR \| GBP \| MXN` | Currency |
| `direction` | `sent \| received` | Direction from user perspective: sent or received |
| `status` | `PENDING \| APPROVED \| REJECTED \| CANCELLED \| PARTIALLY_FULFILLED` | Request status |
| `note` | `string \| null` | Optional note |
| `otherUser` | `object` | The other person in this request |
| `createdAt` | `string` | Creation date |
| `expiresAt` | `string \| null` | Expiration date |

### Approve a money request

```http
POST /v2/money-requests/{requestId}/approve
```


**Path parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `requestId` | `string` | Yes |  |

**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `fromWalletId` | `string` | Yes | Wallet ID to pay from (required) |
| `amount` | `string` | No | Optional amount to pay (for partial fulfillment). If not provided, the full requested amount will be paid. |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` | Exchange rate quote (always present) |
