title: Affiliate & Referrals
description: Referral statistics, referred users, and affiliate tagging.

<Warning>
  The BKey API is in **active development**. Endpoints and schemas may change without notice.
</Warning>


### Get current user referral statistics

```http
GET /affiliate/stats
```


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `referralCount` | `integer` | Number of users referred by this user |
| `referralCode` | `string \| null` | User's referral code, null if not generated yet |
| `earning` | `number` | Total earnings from referrals in USDB |
| `isAffiliate` | `boolean` | Whether the user is an affiliate |


---


### Get current user's referred users with threshold check and pagination

```http
GET /affiliate/referrals
```


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `referredUsers` | `object[]` | List of referred users with threshold check |
| `meta` | `object` | Pagination metadata |


---


### Tag a user as an affiliate (admin only)

```http
POST /affiliate/tag
```


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Whether the user was tagged as affiliate |


---


### Remove a user as an affiliate (admin only)

```http
DELETE /affiliate/remove/{userId}
```


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Whether the affiliate was removed |
