
**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |

### Search contacts by name or phone number

```http
POST /user/contacts/search
```

Search contacts using name (supports first, middle, last names with partial matching) or phone number (with country code). Results ordered by relevance.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `phoneNumber` | `string` | No | Partial phone number with country code for fuzzy matching |
| `name` | `string` | No | Name search with partial matching across first, middle, or last names |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Operation success status |
| `contacts` | `object[]` | List of matching contacts |
| `count` | `number` | Number of contacts found |
| `query` | `string` | The search query used |

### Get mutual contacts for the authenticated user

```http
GET /user/contacts/mutual
```

Returns contacts where both users have added each other (bidirectional relationship). Results are ordered by favourite status first, then by last updated date.


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |

### Sync phone number hashes for contact discovery

```http
POST /user/contacts/sync
```

Processes a batch of phone numbers, creates HMAC-SHA256 hashes, and links them to the authenticated user for contact discovery. Supports up to 3000 phone numbers per request.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `phoneNumbers` | `string[]` | Yes | Array of phone numbers to sync (max 3000) |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | `boolean` | Operation success status |
| `createdPhoneHashes` | `number` | Number of new phone hashes created |
| `createdLinks` | `number` | Number of new user-hash links created |
| `skippedDuplicates` | `number` | Number of duplicate hashes skipped |
| `invalidInputs` | `number` | Number of invalid phone numbers |
| `details` | `object` | Additional operation details |
