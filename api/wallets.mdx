## Wallet

### Create wallet

```http
POST /wallet/v1/wallets
```

Create a wallet after verifying ownership signature. No KYC or partner verification required — fund/withdraw rails are activated separately via onboarding.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `walletAddress` | `string` | Yes | Externally owned wallet address (EOA) |
| `walletIndex` | `integer` | Yes | Wallet index |
| `currency` | `USD \| NGN \| CAD \| EUR \| GBP \| MXN` | Yes | Wallet currency |
| `signature` | `string` | Yes | Signature of the exact message: "I hereby declare that I am the address owner." |
| `colorCode` | `string` | No | Wallet color code (hex). Defaults to #1D6BB9 if omitted. |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `walletId` | `string` | Created wallet ID |
| `walletAddress` | `string` | Original wallet address (EOA) |
| `walletIndex` | `number` | Wallet index |
| `currency` | `USD \| NGN \| CAD \| EUR \| GBP \| MXN` | Wallet currency |
| `colorCode` | `string` | Wallet color code (hex) |
| `isDefault` | `boolean` | Whether this is the user default wallet |
| `canFund` | `boolean` | Whether the wallet can receive fiat funding |
| `canWithdraw` | `boolean` | Whether the wallet can send to fiat rails |


**Query parameters**

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |

**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `wallets` | `object[]` | List of wallets |

## Wallet v2

### Send money to another user (v2)

```http
POST /wallet/v2/send
```

Send money from a wallet to another user. Automatically handles cross-currency conversion if needed.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `fromWalletId` | `string` | Yes | Sender wallet ID |
| `toUserId` | `string` | Yes | Receiver user ID |
| `amount` | `string` | Yes | Amount to send |
| `note` | `string` | No | Optional note for the transfer |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` | Exchange rate quote (present for cross-currency sends) |


---


### Swap between wallets (v2)

```http
POST /wallet/v2/swap
```

Swap currencies between your own wallets. Supports both exactIn (specify input amount) and exactOut (specify desired output amount) modes.


**Request body**

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `fromWalletId` | `string` | Yes | Source wallet ID |
| `toWalletId` | `string` | Yes | Destination wallet ID |
| `swapAmount` | `object \| object` | Yes | Swap amount specification - either exactIn or exactOut |


**Response**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `signatureRequest` | `object` | Signature request details |
| `quote` | `object` | Exchange rate quote for the swap |
